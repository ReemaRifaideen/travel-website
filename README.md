# 🌍 Travel Website

A modern and responsive travel website landing page designed for travelers who love exploring the world. This project focuses on creating an attractive user interface with a dark luxury theme, smooth layouts, and engaging travel sections.

## ✨ Live Demo

🔗 Website: https://ReemaRifaideen.github.io/travel-website/

---

## 📌 About The Project

Travel Website is a front-end web development project created to showcase a travel experience platform.

The website helps users discover destinations, read travel stories, explore travel tips, and get inspired for their next adventure.

The design focuses on:
- Clean modern UI
- Responsive layouts
- Travel-focused visuals
- User-friendly navigation
- Premium dark theme
